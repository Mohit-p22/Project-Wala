Overview

Project-Wala is a web-based platform designed to facilitate the buying and selling of products. It provides a user-friendly interface for sellers to list their items and for buyers to browse, search, and purchase products. The project leverages a combination of frontend and backend technologies to deliver a seamless and interactive user experience.
Features

    User Authentication: Secure login and registration system for both buyers and sellers.
    Product Management: Sellers can add, update, and delete product listings.
    Product Browsing and Searching: Buyers can browse products with various search and filter options.
    Shopping Cart: Allows buyers to add products to their cart and manage their selections before checkout.
  
Technologies Used

    HTML: For structuring the web pages.
    CSS: For styling and layout.
    JavaScript: For client-side scripting and dynamic content.
    Bootstrap: For responsive design and pre-built UI components.
    Java (JDBC): For database connectivity and management.

Installation

To get a local copy up and running, follow these steps:

    Clone the repository:

    sh

git clone https://github.com/Mohit-p22/Project-Wala.git

Navigate to the project directory:

sh

    cd Project-Wala

    Open the project in your preferred code editor.

Usage

    Frontend: Open the index.html file in your web browser to start using the application.
    Backend: Ensure you have a JDBC-compatible database set up. Configure the database connection settings in the Java backend files.
